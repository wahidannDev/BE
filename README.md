# Medpoint Backend

## Introduction
Backend service for Medpoint online reservation system using Raiden (Golang).

## Tech Stack
- Golang
- Raiden Framework
- Supabase (as database and auth provider)

## Folder Structure
- `cmd/` - entry point
- `internal/` - core app logic (controller, service, model, etc)
- `configs/` - config files
- `pkg/` - helper packages
